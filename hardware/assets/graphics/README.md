# Graphics Assets

This folder contains reference artwork, board-art source exports, and rendered
images used during the Replay 2026 Badge hardware design process.

- `temporal/`: vector reference artwork from the released design package.
- `alex/`: working board-art and render assets from the hardware package.

These files are useful for documentation, visual review, and future art
iterations. They are not required for firmware builds.
