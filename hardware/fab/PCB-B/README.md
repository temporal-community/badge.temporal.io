# PCB-B Fabrication Notes

`PCB-B` is the Replay 2026 Badge backplate/art board.

Production specs:

- 1.6 mm, 2-layer PCB.
- Matte black solder mask.
- ENIG finish.

Included outputs:

- `v0-Replay-26_PCB-B_GBR-R0.zip`: Gerber package.
- `v0-Replay-26_PCB-B_BOM-R0.csv`: optional assembly BOM.
- `v0-Replay-26_PCB-B_CPL-R0.csv`: optional assembly placement list.

The BOM and CPL are only needed if the backplate is SMT assembled with Wurth
M2 standoffs. The original production plan skipped that assembly step.

Depanelization is acceptable if it reduces production time and does not change
the finished board outline.
