# External 3D Model References

Some PCB-A footprint model paths pointed at a local library outside the repo.
Those paths were replaced with repo-relative
placeholders under this directory so the KiCad project does not expose local
machine paths.

The original 3D model files were not included. The PCB source,
schematic, footprints, fabrication files, and included board-level STEP exports
remain usable without these optional per-component 3D models.
