Get Started

Core procedures for booting, mounting, configuring, and recovering the badge without digging through a firmware README.

01

Power On

Boot the badge and reach the launcher.

  1. Press the power or reset control once.
  2. Wait for the Temporal splash screen to finish.
  3. Select the launcher from the home screen.
02

Pair With Temporal

Connect the badge to the web experience or event profile.

  1. Open the pairing app on the badge.
  2. Visit temporal.io on your laptop or phone.
  3. Enter the code shown on the badge display.
03

USB Disk Mode

Mount the badge for copying apps, images, firmware, or config files.

  1. Connect USB-C to your computer.
  2. Use the badge bootloader gesture defined by the final hardware.
  3. Copy files to the mounted volume, eject it, then reboot.
04

Load Demo Data

Add a sample workflow history for local demos.

  1. Mount the badge volume.
  2. Copy a JSON trace into the data folder.
  3. Launch Workflow Trace and select the new run.
05

Flash Firmware

Install the current firmware image when a new release is available.

  1. Download the release artifact from the badge repository.
  2. Mount the bootloader volume.
  3. Copy the firmware image and wait for the badge to reboot.
06

Recover

Return a broken configuration to a known-good state.

  1. Back up any user files that still mount.
  2. Replace the config folder with the default release copy.
  3. Reboot and verify the launcher opens.
