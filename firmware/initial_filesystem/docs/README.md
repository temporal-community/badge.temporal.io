# Temporal Badge Docs

These files are embedded into the badge firmware and provisioned onto the
badge filesystem under `/docs`.

  workflow, hardware overview, examples, and gotchas.
- [MicroPython API Reference](API_REFERENCE.md) — exhaustive function-by-function
  reference for the `badge` module and helper libraries.

For production app structure and validation commands, also see
[`/apps/README.md`](../apps/README.md).
